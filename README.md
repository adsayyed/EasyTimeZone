# EasyTimeZone

EasyTimeZone is a simple, user-friendly website designed to help users manage and view time zones with ease. This project consists of a single HTML file, making it lightweight and easy to deploy.

## Overview

EasyTimeZone provides a clean interface for displaying and converting time zones. It is ideal for users who need to quickly check the current time in different locations or coordinate meetings across time zones.

## Features

- View current time in various time zones
- Simple, intuitive interface
- Lightweight and fast (single HTML file)
- No dependencies or installation required
